# Camp Bughaw Landing Page

A modern and responsive landing page built for Camp Bughaw, designed to showcase the camp experience, activities, accommodations, and natural attractions while encouraging visitors to learn more and plan their visit.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Stunning hero section with immersive visuals
* Activities and attractions showcase
* Accommodation highlights
* About Camp Bughaw section
* Photo gallery
* Contact and inquiry section
* Fast-loading and optimized user experience

## Purpose

This project was created to provide Camp Bughaw with a professional online presence that highlights the unique outdoor experiences, scenic views, and recreational activities available to visitors.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Design Principles

## Getting Started

1. Clone the repository
2. Open the project folder
3. Launch `index.html` in your browser

```bash
git clone https://github.com/yourusername/camp-bughaw-landing-page.git
```
